## ğŸ›  Getting Started

### Prerequisites

- Node.js 18+ or Bun
- npm, pnpm, yarn, or bun package manager

### Installation

1. Clone the repository
```bash
git clone https://github.com/YannG75/Portfolio.git
cd Portfolio
```

2. Install dependencies
```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

3. Start the development server
```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev

# bun
bun run dev
```

The site will be available at `http://localhost:3000`

### Build for Production

```bash
# Build the application
npm run build

# Preview the production build locally
npm run preview
```
